Modul de aplicare a unui stil CSS:
  • * - Selector universal
  • div - Selector la nivel de tag
  • .element - Selector la nivel de clasa CSS
  • #deodsebit - Selector la nivel de identificator CSS
Prioritatea aplicarii stilurilor CSS:
  1. stil inline - Selector la nivel de stil CSS inline (style="")
  2. #deodsebit - Selector la nivel de identificator CSS
  3. .element - Selector la nivel de clasa CSS
  4. div - Selector la nivel de tag
  5. * - Selector universal
